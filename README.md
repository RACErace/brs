# BetterSR
BetterSR 是一个可以自动化的工具，支持多账号登录和任务管理。
## 安装

1. 克隆仓库：
    ```sh
    git clone https://github.com/yourusername/BetterSR.git
    cd BetterSR
    ```

2. 安装依赖：
    ```sh
    pip install -r requirements.txt
    ```

## 使用

### 启动应用

运行 `launch.py` 启动应用：
```cmd
python launch.py
```

### 主要功能

- **自动登录**
- **任务管理**
- **每日训练**
